# XNE Assets

**This project is for learning, practice and experiment.**

## Overview

XNE Assets is a C# API use to convert 40+ 3D file format into a custom format that can be loaded into XNEngine.
<br> XNE Assets is powered by [ASSIMP](https://github.com/assimp/assimp/tree/master) by using the [ASSIMP.NET](https://bitbucket.org/Starnick/assimpnet/src/master/).
